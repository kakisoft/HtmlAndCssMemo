background-color: skyblue;

　　box-sizing: border-box;　にしておくと、
　　border や padding を含めて width と heightを計算する。
border: 5px solid yellowgreen;
background-color: tomato;
　　box-sizing: border-box;　を指定しない場合、
　　こんな感じで border の分だけはみ出る。
background-color: orange;
box-shadow: 10px 5px rgba(0, 0, 0, .3);
text-shadow: 10px 5px rgba(0, 0, 0, .3);
　　ぼかし
　　テキストのぼかし
background-color: skyblue;
box-shadow: 10px 5px 3px rgba(0, 0, 0, .3);
text-shadow: 10px 5px 3px rgba(0, 0, 0, .3);
　　　box-shadow の３番目の引数を設定すると、
　　　ぼかし具合を設定できる。
background-color: tomato;
box-shadow: 10px 5px 3px 10px rgba(0, 0, 0, .3);
